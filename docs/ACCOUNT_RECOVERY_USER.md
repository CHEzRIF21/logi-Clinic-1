# Guide Utilisateur - Récupération de Compte

## Quand utiliser la récupération de compte ?

Utilisez cette fonctionnalité si vous avez oublié :
- Votre nom d'utilisateur
- Votre code clinique
- Votre mot de passe

## Comment faire une demande ?

### Étape 1 : Accéder au formulaire

1. Sur la page de connexion, cliquez sur **"Mot de passe oublié ?"** sous le formulaire de connexion
2. Ou allez directement à la section **"Contactez-nous"** en bas de la page
3. Cliquez sur l'onglet **"Récupération de compte"**

### Étape 2 : Remplir le formulaire

Le formulaire demande les informations suivantes :

#### Informations personnelles
- **Code clinique** (optionnel) : Si vous le connaissez, cela accélère le traitement
- **Email** : L'adresse email associée à votre compte
- **Nom** : Votre nom de famille
- **Prénom** : Votre prénom
- **Téléphone** : Votre numéro de téléphone (pour vérification supplémentaire)

#### Questions de sécurité

Vous devez sélectionner et répondre à **3 questions de sécurité** parmi une liste de questions prédéfinies.

**Exemples de questions :**
- Quel est le nom de votre premier animal de compagnie ?
- Dans quelle ville êtes-vous né(e) ?
- Quel est le nom de jeune fille de votre mère ?
- Quel était le nom de votre école primaire ?
- etc.

**Conseils :**
- Choisissez des questions dont vous vous souviendrez facilement
- Répondez exactement comme vous l'avez fait lors de la création de votre compte (si applicable)
- Les réponses sont sensibles à la casse et aux espaces

#### Données à récupérer

Cochez les cases correspondant aux données que vous souhaitez récupérer :
- ☑ Nom d'utilisateur
- ☑ Code clinique
- ☑ Mot de passe (un nouveau mot de passe temporaire sera généré)

### Étape 3 : Soumettre la demande

1. Vérifiez que toutes les informations sont correctes
2. Cliquez sur **"Envoyer la demande"**
3. Un message de confirmation s'affiche

## Que se passe-t-il ensuite ?

### Traitement par l'administrateur

1. **Réception de votre demande** : Un administrateur reçoit une notification
2. **Vérification d'identité** : L'administrateur vérifie vos informations :
   - Compare vos données avec celles de la base de données
   - Vérifie vos réponses aux questions de sécurité (peut vous contacter par téléphone)
3. **Décision** : L'administrateur approuve ou rejette votre demande

### Si votre demande est approuvée

Vous recevrez un **email** contenant :
- Les identifiants demandés (nom d'utilisateur, code clinique si demandé)
- Un **nouveau mot de passe temporaire** (si vous avez demandé la récupération du mot de passe)

**⚠️ IMPORTANT** : Si un nouveau mot de passe vous a été envoyé :
1. Connectez-vous immédiatement avec ce mot de passe temporaire
2. **Changez votre mot de passe** dans les paramètres de votre compte dès votre première connexion
3. Ne partagez jamais votre mot de passe avec qui que ce soit

### Si votre demande est rejetée

Vous recevrez un email vous informant du rejet avec la raison.

**Raisons possibles :**
- Réponses aux questions de sécurité incorrectes
- Informations ne correspondant pas aux données enregistrées
- Utilisateur non trouvé dans la base de données

**Que faire ?**
- Vérifiez que vous avez fourni les bonnes informations
- Contactez le support technique si vous pensez qu'il y a une erreur
- Vous pouvez créer une nouvelle demande après 24 heures

## Délais de traitement

- **Temps de traitement moyen** : 24 à 48 heures
- **Expiration** : Les demandes expirent après 7 jours si non traitées
- **Limite** : Maximum 3 demandes par email par jour

## Conseils pour éviter les problèmes

### Avant de faire une demande

1. **Vérifiez votre boîte email** : Assurez-vous d'avoir accès à l'email associé à votre compte
2. **Préparez vos réponses** : Réfléchissez aux réponses aux questions de sécurité avant de remplir le formulaire
3. **Vérifiez vos informations** : Assurez-vous que le nom, prénom et email correspondent exactement à ceux enregistrés

### Sécurité

- **Ne partagez jamais** vos identifiants avec d'autres personnes
- **Changez votre mot de passe** régulièrement
- **Utilisez un mot de passe fort** : Au moins 8 caractères avec lettres, chiffres et symboles
- **Ne réutilisez pas** vos mots de passe sur d'autres sites

## Questions fréquentes

### Puis-je récupérer mon ancien mot de passe ?

Non, pour des raisons de sécurité, un nouveau mot de passe temporaire est généré. Vous devrez le changer lors de votre prochaine connexion.

### Que faire si je ne me souviens pas des réponses aux questions de sécurité ?

Contactez directement le support technique qui pourra vous aider avec une procédure alternative de vérification d'identité.

### Combien de temps faut-il pour recevoir mes identifiants ?

En général, vous recevrez un email dans les 24 à 48 heures suivant votre demande. En cas d'urgence, contactez le support technique.

### Puis-je faire plusieurs demandes ?

Oui, mais il y a une limite de 3 demandes par email par jour pour des raisons de sécurité.

### Que faire si je ne reçois pas l'email ?

1. Vérifiez votre dossier spam/courrier indésirable
2. Vérifiez que l'adresse email est correcte
3. Attendez 48 heures (délai de traitement)
4. Contactez le support technique
